# Rexo-AI-Chatbot
Rexo is an AI-powered chatbot designed as a portfolio project to demonstrate modern full-stack development and real-world AI integration. It leverages the OpenAI ChatGPT API to deliver intelligent, natural language conversations, allowing users to interact with a responsive and context-aware virtual assistant.
